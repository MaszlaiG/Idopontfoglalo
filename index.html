Időpontfoglaló

Egyetlen, bármelyik weboldalba beágyazható foglalási widget — fodrászatnak, fogászatnak, személyi edzőnek vagy bármilyen szolgáltatónak. Az adatok Firebase-ben tárolódnak, backend fejlesztés nélkül.

Több vállalkozás, egy rendszer

Minden ügyfeled saját azonosítót (business ID) kap, saját szolgáltatásokkal és nyitvatartással.

Iframe beágyazás

Egy sornyi HTML kóddal bármelyik landing oldalba illeszthető, saját színekre hangolható.

Admin felület

A vállalkozás tulajdonosa maga kezeli a foglalásokat, szolgáltatásokat és nyitvatartást.

Ütközésmentes foglalás

A rendszer automatikusan kiszűri a már lefoglalt időpontokat, dupla foglalás nélkül.

Admin felület megnyitása Beüzemelési útmutató
